# 2PG
2PG is a 2-Player Game written in Javascript using Phaser.

# How to play?
To play the game just follow these steps:

```bash
$~ git clone https://github.com/gabru-md/2PG
$~ cd 2PG
$~ firefox index.html
```
You can run it by simply opening the `index.html` file.
